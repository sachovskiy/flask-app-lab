Про себе

Мене звати Саша Сачовський. Я розробник програмного забезпечення з глибоким інтересом до веб-технологій, зокрема Python та Flask. Маю досвід у створенні динамічних веб-застосунків, інтеграції API та налаштуванні серверних рішень. Моя ціль — створювати ефективні, масштабовані та надійні програмні рішення.

Освіта

Закінчив Київський національний університет, де отримав ступінь бакалавра з комп'ютерних наук. Пройшов додаткові курси з веб-розробки та поглибив знання у Python, зокрема в роботі з Flask. Відвідував онлайн-курси з розробки REST API та автоматизації тестування.

Навички

Мої основні навички включають:

  • Python (Flask, Django)
  • JavaScript (React, jQuery)
  • HTML5, CSS3, Bootstrap
  • Git та системи контролю версій
  • SQL та робота з базами даних (PostgreSQL, MySQL)
  • Розробка RESTful API
  • Docker та робота з контейнерами

Досвід роботи

Маю досвід роботи над різними проектами, зокрема:

  • Веб-додаток для керування задачами: Розробка та налаштування backend-частини за допомогою Flask та інтеграція з базою даних PostgreSQL.
  • Платформа для онлайн-навчання: Співпраця з командою у створенні фронтенду за допомогою React та інтеграція з сервером на Django.
  • Особистий блог на Flask: Розробка блогу з можливістю публікації постів, управління користувачами та адміністрування.
